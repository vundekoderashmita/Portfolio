Pneumonia Detection Using CNN and Fuzzy Logic

Project Description and Purpose

  • Built a system to detect pneumonia from chest X-rays.
  • Combined CNN and fuzzy logic for better accuracy and more interpretable decisions.

Technologies Used

  • Dataset: Kaggle X-rays (5216 train, 624 test)
  • Preprocessing: Rescaling + augmentation (flip, zoom, rotate)
  • CNN: Conv + MaxPool + Dropout + Dense, Sigmoid output
  • Fuzzy Logic: Rules — High → Pneumonia, Low → Normal, Medium → Uncertain

Results

  • Test Accuracy: 81.25%
  • Example Outputs:
    • 0.85 → Pneumonia
    • 0.40 → Normal
    • 0.60 → Uncertain
  • CNN handled features well; fuzzy logic improved decision clarity

What I Learned

  • CNN design for medical images
  • Using augmentation to reduce overfitting
  • Fuzzy logic for uncertain cases
  • Value of hybrid models in healthcare AI

Face Detection and Recognition Using Viola-Jones and LBPH

Developed a lightweight facial recognition system using classical computer vision techniques and a small custom dataset.

×

Face Detection and Recognition Using Viola-Jones and LBPH

Project Description and Purpose

  • Built a system to detect and recognize human faces using classical computer vision.
  • The goal was to create a lightweight face recognition tool using a small custom dataset.

Technologies Used

  • Dataset: 75 images (3 people × 25 images)
  • Face Detection: Viola-Jones algorithm using Haar features and cascade classifiers
  • Face Recognition: LBPH (Local Binary Patterns Histogram) method
  • Preprocessing: Grayscale conversion, normalization, enhancement
  • Feature Extraction: LBP patterns + histogram vectors
  • Matching: Euclidean distance for recognition

Results

  • Achieved high accuracy with a small dataset
  • Ratan Tata: 95.27%
  • Narendra Modi: 92.49%
  • Bill Gates: 87.57%
  • Worked well on frontal images under various lighting

What I Learned

  • How Viola-Jones detects faces in real time
  • How LBPH uses texture patterns for recognition
  • Importance of preprocessing and clean datasets
  • Classical CV limitations vs deep learning
  • Built a full pipeline: detection to recognition
